# Website Handmaitie powered by React

## Installation method

1. Clone this Repo

   `git clone https://github.com/Cyrano71/Handamitie.git`
2. Cd into the folder

   `cd Handamitie`
3. Install the dependencies

   `npm install`

We use as dependencies:
- react-redux
- react-router
- framer-motion

4. Start the app

   `npm start`
